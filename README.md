Cara run projek:
0. Pastikan sudah install Node.js
1. Buka terminal di path projek ( ex: C:\Users\M S I\Documents\GitHub\web-pinjam-bt> )
2. Ketik "cd api"
3. Ketik "node index.js"
4. Buka terminal baru, ketik "cd client"
5. Install node_modules dulu jika belum ada, ketik "npm install"
6. Ketik "npm run dev"
7. Masuk ke link yang diberikan
